### Velocity Support
- Added /hub support, configure the default server in config.yml
- Added /cbv broadcast <message>
- Added discord bot support, configure the server name in config.yml if you enable the bot.
  - You need to create multiple bots as you can't share an instance.

### Changes
- Added regex/log filter support which can be configured in server.json.
  - This allows you to remove things you don't want to see in console. (like a certain griefing group)
- Added /cb broadcast <message> command. (/cbv broadcast <message> on Velocity Servers!)
  - You need chatterbox.broadcast to run this permission.
