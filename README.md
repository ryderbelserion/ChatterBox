## Chatter Box

ChatterBox is the kitchen sink of chat formatting for Hytale. You can customize the look of your chat, have integration with LuckPerms, custom join messages, and more to come!

### Installation

The process is simply, You download ChatterBox, and add it to the `mods` folder then restart your server!

Enjoy!
